# FilCan Omni Hunter AI Sales Engine 🚀

**The Relentless Automotive Lead Generation & Management System**

Omni Hunter is a high-performance Full-Stack AI solution designed for automotive dealerships (Tailored for FilCan Cars). It automates the entire lead lifecycle—from first contact on social media to final closing in the CRM.

## 🌟 Key Features

### 🤖 The Omni Hunter Roles
- **Receptionist**: 24/7 Intelligent Greeting and initial lifestyle discovery.
- **Sales Agent**: A relentless 9-step sales script following automotive best practices (Lifestyle, Trade-ins, Financing).
- **Marketing Manager**: AI-generated ad copy and social posts with a **Human-in-the-Loop** approval workflow.

### 🏹 Relentless Hunter Capabilities
- **Dead-Lead Miner**: Automated reactivation of aged leads using intelligent follow-up.
- **Accountability Audit Log**: Transparent, time-stamped tracking of all AI and Human actions.
- **Daily Quality Reporting**: Automated generation of top-tier lead reports with projected ROI.

### 🌐 Omnichannel Demo Suite
- **FilCan Landing Page Clone**: Visualizing the AI as a digital receptionist on the official site.
- **Facebook Clone**: Complete automation of social leads, comment replies, and Messenger engagement.

## 🛠️ Tech Stack
- **Frontend**: React (Vite), Vanilla CSS (Premium Glassmorphism & Modern UI)
- **Backend**: FastAPI (Python), Core AI Logic State Machine
- **Deployment**: Vercel

## 🚀 Getting Started

### Prerequisites
- Python 3.9+
- Node.js 18+

### Installation

1. **Clone the repo:**
   ```bash
   git clone https://github.com/YOUR_USERNAME/ai-sales-engine.git
   cd ai-sales-engine
   ```

2. **Backend Setup:**
   ```bash
   cd backend
   pip install -r requirements.txt
   python -m uvicorn main:app --port 8001 --reload
   ```

3. **Frontend Setup:**
   ```bash
   cd ../frontend
   npm install
   npm run dev
   ```
